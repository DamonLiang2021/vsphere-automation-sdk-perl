Detailed Description

The Com::Vmware::Vcenter::Compute::Policies::Capabilities::Vm_host_affinity::CreateSpec class* contains information used to create a new virtual machine to host affinity policy, see :func:Com::Vmware::Vcenter::Compute::Policies.create . Virtual machines that have the tag indicated by :attr:Com::Vmware::Vcenter::Compute::Policies::Capabilities::Vm_host_affinity::CreateSpec.vm_tag will be affine to hosts that have the tag indicated by :attr:Com::Vmware::Vcenter::Compute::Policies::Capabilities::Vm_host_affinity::CreateSpec.host_tag . Warning: This class is available as technical preview. It may be changed in a future release.
