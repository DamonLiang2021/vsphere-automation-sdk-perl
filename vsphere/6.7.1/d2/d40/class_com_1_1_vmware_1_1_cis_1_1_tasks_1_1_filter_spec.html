Detailed Description

The Com::Vmware::Cis::Tasks::FilterSpec class contains fields used to filter the results when listing tasks (see :func:Com::Vmware::Cis::Tasks.list ). If multiple fields are specified, only tasks matching all of the fields match the filter. This class was added in vSphere API 6.7 U1.
