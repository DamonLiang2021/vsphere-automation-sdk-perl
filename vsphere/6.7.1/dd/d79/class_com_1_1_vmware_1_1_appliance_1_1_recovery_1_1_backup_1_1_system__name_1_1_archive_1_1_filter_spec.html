Detailed Description

The Com::Vmware::Appliance::Recovery::Backup::System_name::Archive::FilterSpec class* contains fields used to filter the results when listing backup archives (see :func:Com::Vmware::Appliance::Recovery::Backup::System_name::Archive.list ). If multiple fields are specified, only backup archives matching all of the fields* match the filter. This class was added in vSphere API 6.7.
