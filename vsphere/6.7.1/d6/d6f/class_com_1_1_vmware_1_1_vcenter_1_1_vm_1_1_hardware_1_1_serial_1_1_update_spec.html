Detailed Description

The Com::Vmware::Vcenter::Vm::Hardware::Serial::UpdateSpec class describes the updates to be made to the configuration of a virtual serial port.

public method Com::Vmware::Vcenter::Vm::Hardware::Serial::UpdateSpec::get_yield_on_poll ( )

Gets the value of 'yield_on_poll' property.

Return values
yield_on_poll- The current value of the field. CPU yield behavior. If set to true, the virtual machine will periodically relinquish the processor if its sole task is polling the virtual serial port. The amount of time it takes to regain the processor will depend on the degree of other virtual machine activity on the host.

This field may be modified at any time, and changes applied to a connected virtual serial port take effect immediately.
