Detailed Description

The Com::Vmware::Esx::Settings::Clusters::Policies::Apply::Effective interface* provides methods to manage the effective policies that will impact how the software and configuration specification documents are applied to an ESXi cluster.

Member Function Documentation

public method Com::Vmware::Esx::Settings::Clusters::Policies::Apply::Effective::get ( )

Returns the effective apply policy based on system defaults and what has been configured for the cluster. Note: Privileges required for this operation are VcIntegrity.lifecycleSettings.Read.

Parameters
cluster[REQUIRED] The cluster identifier. The value must be an identifier for the resource type getQualifiedName(ClusterComputeResource). . The value must be str.
Return values
Theeffective policies that impact the apply method on this cluster. The return type will be Com::Vmware::Esx::Settings::Clusters::Policies::Apply::Effective::EffectivePolicySpec
