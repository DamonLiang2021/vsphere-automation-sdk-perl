Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::TrustAuthorityClusters interface* manages all the Trust Authority Components on each Trust Authority Host in the cluster. The Com::Vmware::Vcenter::Trusted_infrastructure::TrustAuthorityClusters interface transforms a ClusterComputeResource into Trust Authority Cluster and vice versa. This interface* was added in vSphere API 7.0.0.
