Detailed Description

The Com::Vmware::Vcenter::Content::Registries::Harbor::StorageSpec class contains the specification required to configure storage associated with a Harbor registry. In this version, Harbor registry is created in Kubernetes environment, information in this class will result in storage quotas on a Kubernetes namespace.
