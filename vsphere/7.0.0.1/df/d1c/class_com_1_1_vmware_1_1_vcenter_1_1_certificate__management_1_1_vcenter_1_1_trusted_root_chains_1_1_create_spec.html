Detailed Description

The Com::Vmware::Vcenter::Certificate_management::Vcenter::TrustedRootChains::CreateSpec class* contains information to create a trusted root certificate chain. This class* was added in vSphere API 6.7.2.

Gets the value of 'chain' property.

Return values
chain- The current value of the field. Unique identifier for this trusted root. Client can specify at creation as long as it is unique, otherwise one will be generated. An example of a client providing the identifier would be if this trusted root is associated with a VC trust. In this case the identifier would be the domain id.
