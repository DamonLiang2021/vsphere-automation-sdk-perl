Detailed Description

The Com::Vmware::Vcenter::Vcha::Cluster::Witness interface provides methods* to validate a witness's placement configuration and redeploy the witness node in a vCenter High Availability (VCHA) cluster. This interface was added in vSphere API 6.7.1.
