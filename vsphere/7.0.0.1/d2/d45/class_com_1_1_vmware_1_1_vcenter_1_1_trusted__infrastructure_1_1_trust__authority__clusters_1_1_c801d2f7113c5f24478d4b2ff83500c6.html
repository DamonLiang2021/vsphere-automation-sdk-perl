Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trust_authority_clusters::Kms::Providers::ClientCertificate::Info class* contains the client certificate used by the hosts in a cluster for authenticating with the Provider. This class was added in vSphere API 7.0.0.
