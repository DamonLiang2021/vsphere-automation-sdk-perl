Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trusted_clusters::Attestation::Services::Summary class* contains basic information about a registered Attestation Service instance that is configured for a cluster. This class was added in vSphere API 7.0.0.
