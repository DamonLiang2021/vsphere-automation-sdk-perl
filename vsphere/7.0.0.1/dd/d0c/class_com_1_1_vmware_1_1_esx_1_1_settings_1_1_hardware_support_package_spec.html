Detailed Description

The Com::Vmware::Esx::Settings::HardwareSupportPackageSpec class contains fields* to describe the desired Hardware Support Package (HSP) configured for a single device or distinct group of devices (typically the OEM's, including BIOS, device firmware and OEM-supplied driver or agent components).
