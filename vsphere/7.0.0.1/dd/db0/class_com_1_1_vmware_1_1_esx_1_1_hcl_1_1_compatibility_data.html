Detailed Description

This interface provides methods to update the local compatibility data residing on the vCenter Appliance or to get information about the said data. The information in the data is generic VMware compatibility information for servers and devices.

Member Function Documentation

public method Com::Vmware::Esx::Hcl::CompatibilityData::get ( )

Provides information about the compatibility data located on the vCenter Appliance. Note: Privileges required for this operation are VcIntegrity.HardwareCompatibility.Read.

Return values
Informationabout the compatibility data. The return type will be Com::Vmware::Esx::Hcl::CompatibilityData::Status
