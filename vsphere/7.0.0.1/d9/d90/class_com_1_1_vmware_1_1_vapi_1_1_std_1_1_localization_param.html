Detailed Description

This class holds a single message parameter and formatting settings for it. The class* has fields for string , long , double , date time and nested messages. Only one will be used depending on the type of data sent. For date , double* and long it is possible to set additional formatting details. This class* was added in vSphere API 7.0.0.
