Detailed Description

The Com::Vmware::Vcenter::Namespace_management::EdgeClusterCompatibility::FilterSpec class* contains fields used to filter the results when listing Edge Clusters (see :func:Com::Vmware::Vcenter::Namespace_management::EdgeClusterCompatibility.list ) and their compatibility information.
