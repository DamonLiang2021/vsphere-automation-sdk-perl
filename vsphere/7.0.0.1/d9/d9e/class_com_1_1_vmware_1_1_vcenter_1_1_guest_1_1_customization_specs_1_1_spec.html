Detailed Description

The Com::Vmware::Vcenter::Guest::CustomizationSpecs::Spec class contains the specification information and specification object. This is passed to the :func:Com::Vmware::Vcenter::Guest::CustomizationSpecs.set method . This class was added in vSphere API 7.0.0.

Gets the value of 'fingerprint' property.

Return values
fingerprint- The current value of the field. The fingerprint is a unique identifier for a given version of the configuration. Each change to the configuration will update this value. A client cannot change this value. If specified when updating a specification, the changes will only be applied if the current fingerprint matches the specified fingerprint. This field can be used to guard against updates that has happened between the specification content was read and until it is applied. This field was added in vSphere API 7.0.0. String#
