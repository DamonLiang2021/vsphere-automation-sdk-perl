Detailed Description

The Com::Vmware::Vcenter::Identity::Providers::ActiveDirectoryOverLdap class contains the information about to how to use an Active Directory over LDAP connection to allow searching for users and groups if the identity provider is an On-Prem service. This class was added in vSphere API 7.0.0.
