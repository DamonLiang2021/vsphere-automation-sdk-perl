Detailed Description

The Com::Vmware::Vcenter::Lcm::Update::PrecheckReport interface generates precheck report for a vCenter Server instance against a target update version. Constant String::RESOURCE_TYPE # Resource type for precheck report

Member Function Documentation

public method Com::Vmware::Vcenter::Lcm::Update::PrecheckReport::create_task ( )

Creates a vCenter Server pre-update compatibility check report for the pending update version. The report can be exported and downloaded in CSV format.

The result of this operation can be queried by calling the null method where task is the response of this operation.

Parameters
version[REQUIRED] Pending update version for which pre-update compatibility check will be executed. The value must be an identifier for the resource type getQualifiedName(com.vmware.vcenter.lcm.update.pending). . The value must be str.
Return values
Theprecheck report, which contains a link to download the CSV report as well The return type will be string
Exceptions
Com::Vmware::Vapi::Std::Errors::Unauthenticatedif the user can not be authenticated.
Com::Vmware::Vapi::Std::Errors::NotFoundif there is no pending update assosiated with the version in the system.
Com::Vmware::Vapi::Std::Errors::NotAllowedInCurrentStateif a precheck is already in progress.
Com::Vmware::Vapi::Std::Errors::Errorif there is some unknown internal error. The accompanying error message will give more details about the error.
