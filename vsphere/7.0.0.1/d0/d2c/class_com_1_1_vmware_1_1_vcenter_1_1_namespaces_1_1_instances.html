Detailed Description

The Com::Vmware::Vcenter::Namespaces::Instances interface provides methods* to create and delete a namespace object. In this version, an Instance is an abstraction around a Kubernetes namespace. Constant String::RESOURCE_TYPE # The resource type for namespace.
