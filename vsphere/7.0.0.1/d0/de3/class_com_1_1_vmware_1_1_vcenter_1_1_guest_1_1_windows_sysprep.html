Detailed Description

The Com::Vmware::Vcenter::Guest::WindowsSysprep class provides all the settings like user details, administrator details, etc that need to applied for a windows guest operating system during customization. This class was added in vSphere API 7.0.0.

Gets the value of 'gui_run_once_commands' property.

Return values
gui_run_once_commands- The current value of the field. A list of commands to run at first user logon, after customizing the guest. These commands are directly mapped to the GuiRunOnce key in the sysprep.xml answer file. These commands are transferred into the sysprep.xml file that VirtualCenter stores on the target virtual disk. For more information about performing unattended installation, check https://technet.microsoft.com/en-us/library/cc771830(v=ws.10).aspx The commands listed here ar executed when a user logs on the first time after customization completes. The logon may be driven by :attr:Com::Vmware::Vcenter::Guest::GuiUnattended.auto_logon setting. These commands are directly mapped to the GuiRunOnce key in the.
