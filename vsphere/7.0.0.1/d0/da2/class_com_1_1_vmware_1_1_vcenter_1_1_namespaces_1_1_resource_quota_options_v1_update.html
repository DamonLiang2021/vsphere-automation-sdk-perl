Detailed Description

The Com::Vmware::Vcenter::Namespaces::ResourceQuotaOptionsV1Update class represents the changes to resource quota limits which are set on the namespace. Refer to < Kubernetes Resource Quota for information related to the properties of this object and what they map to.
