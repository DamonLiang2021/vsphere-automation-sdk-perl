Detailed Description

A set of timestamps and statistical values representing a time series. The lengths of :attr:Com::Vmware::Vcenter::Namespace_management::Stats::TimeSeries::TimeSeries.time_stamps and :attr:Com::Vmware::Vcenter::Namespace_management::Stats::TimeSeries::TimeSeries.values will always match each other.
