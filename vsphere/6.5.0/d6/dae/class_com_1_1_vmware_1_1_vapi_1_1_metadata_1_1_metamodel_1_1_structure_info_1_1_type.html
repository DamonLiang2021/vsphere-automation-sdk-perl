Detailed Description

The Com::Vmware::Vapi::Metadata::Metamodel::StructureInfo::Type enumerated type defines the kind of this structure element. In the interface definition language, structure element and error element have similar characteristics. The difference is that only error elements can be used to describe the errors of an operation element. Constant Com::Vmware::Vapi::Metadata::Metamodel::StructureInfo::Type::STRUCTURE # If the type is a structure element. Constant Com::Vmware::Vapi::Metadata::Metamodel::StructureInfo::Type::ERROR # If the type is an error element.
