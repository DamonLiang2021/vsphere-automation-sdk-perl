Detailed Description

The Com::Vmware::Appliance::Networking::Interfaces::Ipv6::Config class provides the structure that you can use to configure IPv6 on a particular interface. Addresses can be assigned by DHCP, SLAAC or STATIC, as IPv6 permits multiple addresses per interface. This class was added in vSphere API 6.7

public method Com::Vmware::Appliance::Networking::Interfaces::Ipv6::Config::get_default_gateway ( )

Gets the value of 'default_gateway' property.

Return values
default_gateway- The current value of the field. The default gateway for static IP address assignment. This configures the global IPv6 default gateway on the appliance with the specified gateway address and interface. This gateway replaces the existing default gateway configured on the appliance. However, if the gateway address is link-local, then it is added for that interface. This does not support configuration of multiple global default gateways through different interfaces. This field was added in vSphere API 6.7 String#
