Detailed Description

The Com::Vmware::Vcenter::Storage::Policies::VM interface provides methods* managing the storage policy association for a virtual machine and its virtual disks. This interface was added in vSphere API 6.7

Member Function Documentation

public method Com::Vmware::Vcenter::Storage::Policies::VM::list ( )

Returns information about the virtual machines and/or their virtual disks that are associated with the given storage policy. This method was added in vSphere API 6.7

Parameters
policy[REQUIRED] storage policy identifier. The value must be an identifier for the resource type getQualifiedName(com.vmware.vcenter.StoragePolicy). . The value must be str.
Return values
Informationabout the virtual machines and/or their virtual disks that are associated with the given storage policy. The value will be an identifier for the resource type getQualifiedName(VirtualMachine). The return type will be Dictionary of str and Com::Vmware::Vcenter::Storage::Policies::VM::Info
Exceptions
Com::Vmware::Vapi::Std::Errors::NotFoundif there is no policy associated with policy in the system.
Com::Vmware::Vapi::Std::Errors::UnableToAllocateResourceif more than 1000 virtual machines are associated with the specified policy.
Com::Vmware::Vapi::Std::Errors::Unauthenticatedif the user cannot be authenticated.
Com::Vmware::Vapi::Std::Errors::ServiceUnavailableif the system is unable to communicate with a service necessary to complete the request.
Com::Vmware::Vapi::Std::Errors::Errorif the backend server encounters some an error while processing the request.
Com::Vmware::Vapi::Std::Errors::Unauthorizedif the user does not have the required priveleges.
