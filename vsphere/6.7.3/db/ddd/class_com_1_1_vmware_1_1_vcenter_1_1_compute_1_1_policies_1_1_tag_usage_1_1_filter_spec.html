Detailed Description

The Com::Vmware::Vcenter::Compute::Policies::TagUsage::FilterSpec class contains fields used to filter the results when listing the tags used by policies as available in this vCenter server (see :func:Com::Vmware::Vcenter::Compute::Policies::TagUsage.list ). If multiple fields* are specified, only the tags used by policies that match an element of each field* match the filter. Warning: This class is available as technical preview. It may be changed in a future release.
