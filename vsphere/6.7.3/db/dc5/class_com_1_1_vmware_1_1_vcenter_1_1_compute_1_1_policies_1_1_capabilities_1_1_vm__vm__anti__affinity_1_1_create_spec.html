Detailed Description

The Com::Vmware::Vcenter::Compute::Policies::Capabilities::Vm_vm_anti_affinity::CreateSpec class* contains information used to create a new VM-VM anti-affinity policy, see :func:Com::Vmware::Vcenter::Compute::Policies.create . All virtual machines that share the tag indicated by :attr:Com::Vmware::Vcenter::Compute::Policies::Capabilities::Vm_vm_anti_affinity::CreateSpec.vm_tag will be anti-affine to each other. Warning: This class is available as technical preview. It may be changed in a future release.
