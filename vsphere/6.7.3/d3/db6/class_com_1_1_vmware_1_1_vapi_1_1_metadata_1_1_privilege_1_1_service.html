Detailed Description

The Com::Vmware::Vapi::Metadata::Privilege::Service interface provides methods* to retrieve privilege information of a service element.

A service element is said to contain privilege information if one of the operation elements contained in this service element has privilege information.

Constant String::RESOURCE_TYPE # Resource type for service.
