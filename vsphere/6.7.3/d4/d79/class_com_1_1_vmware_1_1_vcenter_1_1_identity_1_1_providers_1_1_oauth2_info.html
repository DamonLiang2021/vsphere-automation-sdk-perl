public method Com::Vmware::Vcenter::Identity::Providers::Oauth2Info::get_auth_query_params ( )

Gets the value of 'auth_query_params' property.

Return values
auth_query_params- The current value of the field.

key/value pairs that are to be appended to the authEndpoint request.

How to append to authEndpoint request:

If the map is not empty, a "?" is added to the endpoint URL, and combination of each k and each string in the v is added with an "&" delimiter. Details:

  • If the value contains only one string, then the key is added with "k=v".
  • If the value is an empty list, then the key is added without a "=v".
  • If the value contains multiple strings, then the key is repeated in the query-string for each string in the value.

. Warning: This field is available as technical preview. It may be changed in a future release. Map#

public method Com::Vmware::Vcenter::Identity::Providers::Oauth2Info::get_claim_map ( )

Gets the value of 'claim_map' property.

Return values
claim_map- The current value of the field. The map used to transform an OAuth2 claim to a corresponding claim that vCenter Server understands. Currently only the key "perms" is supported. The key "perms" is used for mapping the "perms" claim of incoming JWT. The value is another map with an external group as the key and a vCenter Server group as value.
