Detailed Description

The Com::Vmware::Vcenter::Vcha::Operations interface provides methods to figure out the currently active operations and currently disabled operations in a vCenter High Availability (VCHA) cluster. public method Com::Vmware::Vcenter::Vcha::Operations::get ( )

Retrieves the current active and disabled operations of a VCHA cluster. This method was added in vSphere API 6.7.1.

Return values
Infostructure containing the current running and disabled operations of a VCHA cluster. The return type will be Com::Vmware::Vcenter::Vcha::Operations::Info
