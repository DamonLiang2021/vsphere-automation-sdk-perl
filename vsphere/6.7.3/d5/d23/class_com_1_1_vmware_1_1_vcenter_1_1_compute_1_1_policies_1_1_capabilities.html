Detailed Description

The Com::Vmware::Vcenter::Compute::Policies::Capabilities interface provides methods* to manage compute policy capabilities. The description of the capability provides information about the intent of a policy based on this capability. A capability provides a type to create a policy (see :func:Com::Vmware::Vcenter::Compute::Policies.create ). A capability also provides a type that describes the information returned when retrieving information about a policy (see :func:Com::Vmware::Vcenter::Compute::Policies.get ). Warning: This interface is available as technical preview. It may be changed in a future release. Constant String::RESOURCE_TYPE # The resource type for the compute policy capability. Warning: This constant is available as technical preview. It may be changed in a future release.
