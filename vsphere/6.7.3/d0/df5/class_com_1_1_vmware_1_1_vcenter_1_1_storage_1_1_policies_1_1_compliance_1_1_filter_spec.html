Detailed Description

The Com::Vmware::Vcenter::Storage::Policies::Compliance::FilterSpec class contains complianceStatus used to filter the results when listing entities (see :func:Com::Vmware::Vcenter::Storage::Policies::Compliance.list ). This class was added in vSphere API 6.7.
