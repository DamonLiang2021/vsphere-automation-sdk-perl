Detailed Description

The Com::Vmware::Vcenter::Hvc::Management::Administrators provides methods to update, delete, and list groups in the local sso group. This is limited to the Hybrid Linked Mode service. Warning: This interface is available as technical preview. It may be changed in a future release.
