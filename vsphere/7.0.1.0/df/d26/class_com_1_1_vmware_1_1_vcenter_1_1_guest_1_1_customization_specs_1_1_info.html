Detailed Description

The Com::Vmware::Vcenter::Guest::CustomizationSpecs::Info class describes a guest customization specification and the timestamp when it was last modified. This is returned by the :func:Com::Vmware::Vcenter::Guest::CustomizationSpecs.get method* . This class was added in vSphere API 7.0.0.0.
