Detailed Description

The Com::Vmware::Vstats::Counters interface provides methods to perform various Counter related operations. Counter is derived from metric. It applies the metric to a particular class of a resource. Warning: This interface is available as Technology Preview. These are early access APIs provided to test, automate and provide feedback on the feature. Since this can change based on feedback, VMware does not guarantee backwards compatibility and recommends against using them in production environments.
