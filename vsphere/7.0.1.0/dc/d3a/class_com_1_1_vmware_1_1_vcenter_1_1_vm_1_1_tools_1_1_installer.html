Detailed Description

The Com::Vmware::Vcenter::Vm::Tools::Installer ( service} provides methods* to install VMware Tools in the guest operating system. This interface was added in vSphere API 7.0.0.0.

Member Function Documentation

public method Com::Vmware::Vcenter::Vm::Tools::Installer::connect ( )

Connects the VMware Tools CD installer as a CD-ROM for the guest operating system. On Windows guest operating systems with autorun, this should cause the installer to initiate the Tools installation which will need user input to complete. On other (non-Windows) guest operating systems this will make the Tools installation available, and a a user will need to do guest-specific actions. On Linux, this includes opening an archive and running the installer. To monitor the status of the Tools install, clients should check the versionStatus and runState from :func:Com::Vmware::Vcenter::Vm::Tools.get . This method was added in vSphere API 7.0.0.0.
