Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trust_authority_clusters::Attestation::Tpm2::EndorsementKeys interface* provides methods to manage Trusted Platform Module (TPM) Endorsement Keys (EK) on a cluster level. This interface was added in vSphere API 7.0.0.0. Constant String::RESOURCE_TYPE # Resource type for TPM 2.0 endorsement keys. This constant was added in vSphere API 7.0.0.0.
