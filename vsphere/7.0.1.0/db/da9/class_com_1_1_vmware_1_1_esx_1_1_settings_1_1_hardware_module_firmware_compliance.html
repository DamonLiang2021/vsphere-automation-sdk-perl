Detailed Description

The Com::Vmware::Esx::Settings::HardwareModuleFirmwareCompliance class contains information that describe the compliance of firmware of a particular hardware module (e.g. BIOS, PCI device) on the host with respect to the firmware present in the target Hardware Support Package (HSP) specification.
