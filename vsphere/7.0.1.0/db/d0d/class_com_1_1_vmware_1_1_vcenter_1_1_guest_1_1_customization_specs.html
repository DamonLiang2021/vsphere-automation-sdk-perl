Detailed Description

The Com::Vmware::Vcenter::Guest::CustomizationSpecs interface provides methods* to manage guest customization specifications in the vCenter Server. This interface* was added in vSphere API 6.7.1. Constant String::RESOURCE_TYPE # The resource type for a vCenter guest customization specification. This constant was added in vSphere API 6.7.1.
