Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trusted_clusters::Kms::Services::CreateSpec class* contains the data necessary for configuring a registered Key Provider Service instance with a cluster in the environment. This class was added in vSphere API 7.0.0.0.
