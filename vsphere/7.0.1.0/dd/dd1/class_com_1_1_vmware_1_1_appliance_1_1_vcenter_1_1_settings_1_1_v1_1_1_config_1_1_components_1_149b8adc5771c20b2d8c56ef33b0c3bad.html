Detailed Description

The Com::Vmware::Appliance::Vcenter::Settings::V1::Config::Components::Applmgmt::RecurrenceInfo::DayOfWeek enumerated type* defines the set of days when backup can be scheduled. The days can be specified as a list of individual days. You specify the days when you set the recurrence for a schedule.
