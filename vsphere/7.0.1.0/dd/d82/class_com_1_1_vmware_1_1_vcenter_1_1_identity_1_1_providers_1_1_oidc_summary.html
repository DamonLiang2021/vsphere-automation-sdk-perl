Detailed Description

The Com::Vmware::Vcenter::Identity::Providers::OidcSummary class contains commonly used information about an OIDC identity provider. OIDC is a discovery protocol for OAuth2 configuration metadata, so Com::Vmware::Vcenter::Identity::Providers::OidcSummary contains discovered OAuth2 metadata. This class was added in vSphere API 7.0.0.0.

public method Com::Vmware::Vcenter::Identity::Providers::OidcSummary::get_auth_query_params ( )

Gets the value of 'auth_query_params' property.

Return values
auth_query_params- The current value of the field.

key/value pairs that are to be appended to the authEndpoint request.

How to append to authEndpoint request:

If the map is not empty, a "?" is added to the endpoint URL, and combination of each k and each string in the v is added with an "&" delimiter. Details:

  • If the value contains only one string, then the key is added with "k=v".
  • If the value is an empty list, then the key is added without a "=v".
  • If the value contains multiple strings, then the key is repeated in the query-string for each string in the value.

. This field was added in vSphere API 7.0.0.0. Map#
