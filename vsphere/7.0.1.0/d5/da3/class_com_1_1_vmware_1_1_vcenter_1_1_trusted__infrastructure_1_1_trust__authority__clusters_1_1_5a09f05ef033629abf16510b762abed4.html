Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trust_authority_clusters::ConsumerPrincipals::CreateSpec class* contains the information necessary to establish trust between a workload vCenter and a Trust Authority Host. This class was added in vSphere API 7.0.0.0.
