Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trusted_clusters::Kms::ServicesAppliedConfig::Summary class* contains basic information about the aggregated health status for a service. This class was added in vSphere API 7.0.1.0.
