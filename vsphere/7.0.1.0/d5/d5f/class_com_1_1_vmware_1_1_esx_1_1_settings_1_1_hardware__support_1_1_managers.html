Detailed Description

The Com::Vmware::Esx::Settings::Hardware_support::Managers interface provides methods to list Hardware Support Manager (HSM) for a given vCenter.

Member Function Documentation

public method Com::Vmware::Esx::Settings::Hardware_support::Managers::list ( )

Returns the list of registered Hardware Support Managers (HSMs) in the system. Note: Privileges required for this operation are VcIntegrity.lifecycleSettings.Read.

Return values
Listof currently registered Hardware Support Manager (HSMs) The return type will be Array of Com::Vmware::Esx::Settings::Hardware_support::Managers::HardwareSupportManagerInfo
