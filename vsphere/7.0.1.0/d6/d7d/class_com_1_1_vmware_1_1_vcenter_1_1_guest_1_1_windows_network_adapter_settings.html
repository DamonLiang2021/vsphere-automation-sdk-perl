Detailed Description

The Com::Vmware::Vcenter::Guest::WindowsNetworkAdapterSettings class specifies Windows specific Network settings for a virtual network adapter. This class* was added in vSphere API 7.0.0.0.

Gets the value of 'dns_servers' property.

Return values
dns_servers- The current value of the field. A list of server IP addresses to use for DNS lookup in a Windows guest operating system.

These servers should be specified in the order of preference. If this list is not empty, and if a DHCP IP address is used, then these settings override the DHCP settings.
