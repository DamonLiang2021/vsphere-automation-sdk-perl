Detailed Description

Com::Vmware::Vcenter::Namespace_management::LoadBalancers::HAProxyConfigCreateSpec captures the configuration data required for Supervisor Service Type:LoadBalancer to have an external load balancer be created via HAProxy. This class was added in vSphere API 7.0.1.0.
