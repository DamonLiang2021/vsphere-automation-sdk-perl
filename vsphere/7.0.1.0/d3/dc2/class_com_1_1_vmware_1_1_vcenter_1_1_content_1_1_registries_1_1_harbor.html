Detailed Description

The Com::Vmware::Vcenter::Content::Registries::Harbor interface provides methods* to manage the lifecycle of an integrated Harbor container registry in vCenter.

Creates a Harbor registry in the cluster.

Parameters
client_token[OPTIONAL] A unique token generated on the client for each creation request.
