Detailed Description

The Com::Vmware::Vcenter::Namespace_management::Clusters::NCPClusterNetworkEnableSpec class* encapsulates the NSX Container Plugin-specific cluster networking configuration parameters for the vSphere Namespaces Cluster Enable operation.
