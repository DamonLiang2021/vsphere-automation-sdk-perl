Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trust_authority_clusters::Attestation::Tpm2::Settings interface* provides methods to get or update settings related to the TPM 2.0 attestation protocol behavior. This interface was added in vSphere API 7.0.0.0.
