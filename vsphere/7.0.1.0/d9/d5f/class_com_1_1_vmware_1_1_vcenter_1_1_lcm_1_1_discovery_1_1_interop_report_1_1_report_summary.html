Detailed Description

The Com::Vmware::Vcenter::Lcm::Discovery::InteropReport::ReportSummary class contains a summary of the :attr:Com::Vmware::Vcenter::Lcm::Discovery::InteropReport::Report.products . It consists of the count of compatible and incompatible products to the target product.
