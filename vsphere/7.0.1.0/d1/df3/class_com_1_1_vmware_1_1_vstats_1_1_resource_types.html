Detailed Description

The Com::Vmware::Vstats::ResourceTypes interface provides list of resource types. Resource refers to any item or concept that could have measurable properties. It is a prerequisite that a resource can be identified.

Example resource types are virtual machine, virtual disk etc.

. Warning: This interface is available as Technology Preview. These are early access APIs provided to test, automate and provide feedback on the feature. Since this can change based on feedback, VMware does not guarantee backwards compatibility and recommends against using them in production environments.
