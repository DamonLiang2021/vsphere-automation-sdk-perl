Detailed Description

This interface provides methods to download information generated from the hardware compatibility feature residing on the vCenter Appliance.

Member Function Documentation

public method Com::Vmware::Esx::Hcl::Reports::get ( )

Returns the location class Com::Vmware::Esx::Hcl::Reports::Location information for downloading a compatibility report. Note: Privileges required for this operation are VcIntegrity.HardwareCompatibility.Read.

Parameters
report[REQUIRED] identifier of hardware compatiblity report to be downloaded. The value must be an identifier for the resource type getQualifiedName(com.vmware.esx.hcl.resources.CompatibilityReport). . The value must be str.
Return values
