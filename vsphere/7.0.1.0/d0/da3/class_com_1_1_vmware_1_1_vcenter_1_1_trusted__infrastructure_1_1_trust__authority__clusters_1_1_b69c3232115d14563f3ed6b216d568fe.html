Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trust_authority_clusters::Kms::Providers::CurrentPeerCertificates::FilterSpec class* contains fields used to filter the results when listing remote server certificates. This class was added in vSphere API 7.0.0.0.
