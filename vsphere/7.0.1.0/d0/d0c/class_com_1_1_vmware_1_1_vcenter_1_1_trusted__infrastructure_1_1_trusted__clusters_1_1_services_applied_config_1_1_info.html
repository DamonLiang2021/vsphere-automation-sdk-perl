Detailed Description

The Com::Vmware::Vcenter::Trusted_infrastructure::Trusted_clusters::ServicesAppliedConfig::Info class* contains detailed information about the health of the applied Trust Authority Component configurations in a cluster. This class was added in vSphere API 7.0.1.0.
