Detailed Description

The Com::Vmware::Vcenter::Namespace_management::SupportBundle::Location class contains the URI location to download the per-cluster support bundle from, as well as a token required (as a header on the HTTP request) to get the bundle. The validity of the token is 5 minutes. After the token expires, any attempt to call the URI with said token will fail.
