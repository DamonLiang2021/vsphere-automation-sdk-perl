Detailed Description

The Com::Vmware::Vcenter::Certificate_management::Vcenter::VmcaRoot interface provides methods to replace VMware Certificate Authority (VMCA) root certificate. This interface was added in vSphere API 6.9.1.

Member Function Documentation

public method Com::Vmware::Vcenter::Certificate_management::Vcenter::VmcaRoot::create ( )

Replace Root Certificate with VMCA signed one using the given Spec.

After this method completes, the services using the certificate will be restarted for the new certificate to take effect.

. This method was added in vSphere API 6.9.1. Note: Privileges required for this operation are CertificateManagement.Administer.

Parameters
spec[OPTIONAL] The information needed to generate VMCA signed Root Certificate. Default values will be set for all null parameters. . The value must be Com::Vmware::Vcenter::Certificate_management::Vcenter::VmcaRoot::CreateSpec or None.
Exceptions
Com::Vmware::Vapi::Std::Errors::ErrorIf the system failed to renew the TLS certificate.
