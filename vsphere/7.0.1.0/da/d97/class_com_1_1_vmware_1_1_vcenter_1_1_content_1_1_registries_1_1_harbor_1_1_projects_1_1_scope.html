Detailed Description

The Com::Vmware::Vcenter::Content::Registries::Harbor::Projects::Scope enumerated type* in a project defines access levels of the project. Constant Com::Vmware::Vcenter::Content::Registries::Harbor::Projects::Scope::PUBLIC # A Harbor project can be accessed by everyone. Constant Com::Vmware::Vcenter::Content::Registries::Harbor::Projects::Scope::PRIVATE # A Harbor project can only be accessed by assigned users.
